How Predictive Analysis is Calculated

The predictive chart uses simple linear regression to forecast future population values for each region or province. This method fits a straight line to the historical data and extends it to predict future values.

  • For each region or province, the chart takes the historical population data (e.g., 2020–2025).
  • It converts the years to numbers (e.g., 2020, 2021, ...).
  • It fits a line y = mx + b using the least squares method, where y is population and x is year.
  • The slope (m) and intercept (b) are calculated from the data.
  • The model predicts future values by extending the line for the chosen number of years ahead.

Linear Regression Formula:
y = m x + b
where
m = (N Σxy - Σx Σy) / (N Σx² - (Σx)²)
b = (Σy - m Σx) / N
N is the number of data points, x is year, y is population.

This approach is simple and works well for data with a roughly linear trend. For more advanced forecasting, more complex models can be used.
