How Descriptive Analysis is Performed

Descriptive analysis summarizes and visualizes the main features of a dataset, providing insights into its structure and trends. For the Philippine population dataset used in this dashboard, descriptive analysis involves:

  • Data Cleaning: Ensuring province and region names are consistent, handling missing or anomalous values, and standardizing year columns.
  • Aggregation: Summing population values by region, province, or year to reveal overall and group-level trends.
  • Trend Analysis: Examining how population changes over time for each province and region using line, bar, and area charts.
  • Distribution Analysis: Using pie, doughnut, and polar area charts to show how the population is distributed among provinces within a region for a given year.
  • Comparative Analysis: Comparing population values across provinces, regions, or years using stacked bar, radar, and bubble charts.
  • Visualization: Presenting the data in various chart types to make patterns, outliers, and relationships easy to see and interpret.

  • For a radar chart, compare the population of several provinces in a region for a single year.

Descriptive analysis does not predict the future but helps you understand the current and historical state of the data. It is the foundation for deeper statistical or predictive analysis.
